которая выводит триугольник со сторонами длины `length`:
```c
#include <stdio.h>

void printTriangle(int length) {
    // ...
}

int main() {
    printTriangle(2);
    printTriangle(3);
    printTriangle(4);
}
```

```
*
**
*
**
***
*
**
***
****
```
